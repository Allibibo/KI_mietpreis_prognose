Wie kann ich das Tool verwenden?

1. Öffne die Datei: Eingabemaske.ipynb
2. Führe das das Jupiter File aus und gib die gewünschten Eigenschaften in der Eingabemaske ein


Hinweis:

In Jupyter Notebooks können Sie einzelne Codeblöcke separat ausführen. 
Allerdings sollten Sie beachten, dass einige dieser Ausführungen erhebliche Rechenzeit 
in Anspruch nehmen können. Es empfiehlt sich, zuerst durch die Notebooks zu blättern und 
sich die Visualisierungen anzuschauen. Beachten Sie, dass die Diagramme erst wieder angezeigt werden,
wenn Sie die zugehörigen Blöcke erneut ausführen.
